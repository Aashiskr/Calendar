# Interactive Wall Calendar

A responsive, interactive wall calendar component built with Next.js, TypeScript, and Framer Motion.

## Features

- Physical wall calendar look with nail, spiral binding and page-flip animations
- 12 unique monthly hero images with seasonal themes
- Day range selection with visual feedback
- Notes system with time support (persisted in localStorage)
- Built-in calendar assistant chatbot
- Light/dark theme toggle with textured wall backgrounds
- Fully responsive design

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **CSS Modules**
- **Framer Motion** for animations
- **Lucide React** for icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
src/
├── app/            # Next.js app router pages
├── components/     # React components
├── data/           # Static data (holidays, month metadata)
├── hooks/          # Custom hooks (calendar, notes, theme, selection)
└── styles/         # CSS Modules
```
